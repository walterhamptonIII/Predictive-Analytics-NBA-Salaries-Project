This project analyzes the 2020-21 season to isolate "performance-driven value" from contractual noise (like rookie scales and veteran minimums).🛠️ Tech StackLanguage: Python 3.11+Libraries: Scikit-Learn, Pandas, NumPy, Plotly (Interactive Viz)Models: Random Forest, Gradient Boosting, Ridge Regression🧬 Data Engineering & PipelineTo ensure high model integrity, I implemented a robust preprocessing pipeline:Position Normalization: Handled hybrid roles (e.g., PF-C normalized to PF) using string-split logic to ensure categorical consistency.Data Leakage Prevention: Scaled features using a Pipeline to ensure the model only learned from training distributions.Feature Selection: Explicitly removed "leaking" variables like guaranteed_money and salary_rank to ensure the model relied purely on box-score statistics.📊 Modeling & ResultsI benchmarked three distinct mathematical approaches to identify the most accurate predictor of compensation:ModelR² ScoreMAE (Mean Error)NoteRandom Forest0.58$3.39MBest at capturing non-linear "Superstar" effects.Gradient Boosting0.51$3.51MHigh precision on mid-tier player valuations.Ridge Regression0.35$4.20MBaseline linear model using L2 regularization.💡 Key InsightsPrimary Drivers: While PTS (Points) is the highest linear correlation, the Random Forest identified Age and Minutes Played as the most critical weights for predicting actual salary.The Rookie Scale Gap: Diagnostic plots revealed a significant cluster of "high-performance, low-pay" players, highlighting the impact of rookie-scale contracts on league-wide valuation.🚀 How to RunClone the repository.Install dependencies: pip install -r requirements.txtOpen NBA project second draft.ipynb to view the interactive Plotly visualizations.
